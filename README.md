# Tareas curso EDV - Programación cientifica para proyectos de IA
Repositorio para subir las tareas realizadas durante la Escuela de Verano en el curso "Programación científica para proyectos de Inteligencia Artificial" del 2026.

La primera tarea está enfocada en el análisis exploratorio de datos, llevado a cabo por medio del análisis del dataset 'diamonds', en donde es evaluado nuestro uso correcto de las herramientas del lenguaje de programación Python y nuestras capacidades para realizar un buen análisis estadístico-descriptivo y una eficiente concepción de estrategias para transmitir información a través de visualizaciones apropiadas. Asímismo, también se grafica y se interpretan las relaciones de las variables con el precio y cómo influyen en su comportamiento.
